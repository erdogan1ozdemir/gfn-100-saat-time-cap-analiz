# GeForce NOW 100 Saat Sınırı - Türkiye Kullanıcı Algısı Panosu

GAME+ (Turkcell) GeForce NOW'da 10 Haziran 2026'da başlayan aylık 100 saat sınırına Türk kullanıcıların tepkisini ölçen tek dosyalık sosyal dinleme panosu.

- **Tek HTML dosyası**, dış bağımlılık yok, internetsiz açılır.
- **Açık / koyu mod** (sağ alttaki düğme; tercih tarayıcıda hatırlanır).
- Kaynaklar: Technopat, DonanımHaber, Ekşi Sözlük, Şikayetvar, X/Twitter, YouTube (DataForSEO), Reddit (Google üzerinden); 3 yapay zeka raporu doğrulanıp birleştirildi.
